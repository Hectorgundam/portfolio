Maze Traversal

A class assignment to implement a maze traversal algorithm.

For this assignment, the user the program asks the user to enter a .txt labyrinth file with preset characters to represent walls, paths, start and end points. The program then finds the shortest path through the labyrinth and outputs the path taken and the number of steps. The program then generates a VRML file to visualize the solution.
