Permutations Using Binary Nodes

Java program that calculates and generates all possible permutations for a given string using recursive binary node logic.

Code Highlight: Recursive Permutation Logic

public class GenStr {
    private String source;
    private Node root;

    public void generatePermutations(String value) {
        source = value;
        root = new Node("");
        buildTree(root, 0);
    }

    private void buildTree(Node current, int index) {
        if (index == source.length()) {
            System.out.println(current.value);
            return;
        }

        buildTree(new Node(current.value + source.charAt(index)), index + 1);
        buildTree(new Node(current.value), index + 1);
    }
}
